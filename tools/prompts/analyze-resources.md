# Prompt: Analyze AWS Resources

Use this prompt with the AI Assistant to get a comprehensive analysis.

## Full Infrastructure Analysis
```
전체 AWS 인프라 현황을 분석하고, 보안 이슈와 비용 최적화 제안을 포함해줘
```

## Network Troubleshooting
```
VPC01의 i-xxx에서 DMZ VPC의 i-yyy로 TCP 443 통신이 가능한지
라우트 테이블과 보안그룹을 분석해줘
```

## Security Audit
```
현재 보안 이슈를 종합적으로 분석해줘:
- 퍼블릭 S3 버킷
- 0.0.0.0/0 오픈된 보안그룹
- 암호화되지 않은 EBS 볼륨
- IAM 사용자 현황
```

## Cost Analysis
```
현재 비용 현황을 서비스별로 분석하고,
비용 절감 방안을 제안해줘
```

## Code Execution
```
EC2 인스턴스 타입별 비용 효율을 계산하는 코드 실행해줘
```
