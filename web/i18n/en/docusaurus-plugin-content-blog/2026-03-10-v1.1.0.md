A milestone release featuring a complete redesign from a single Gateway (29 tools) to **7 specialized Gateways x 125 MCP tools**.

<!--truncate-->

### 7 Role-Based Gateways

Each Gateway provides a tool set optimized for a specific domain.

| Gateway | Tools | Key Features |
|---------|-------|--------------|
| Infra | 41 | Network, EKS, ECS, Istio |
| IaC | 12 | CloudFormation, CDK, Terraform |
| Data | 24 | DynamoDB, RDS, ElastiCache, MSK |
| Security | 14 | IAM users/roles/policies/simulation |
| Monitoring | 16 | CloudWatch, CloudTrail |
| Cost | 9 | Cost Explorer, Budgets |
| Ops | 9 | Steampipe SQL, AWS documentation |

### 19 Lambda Functions

Lambda functions providing MCP tools.

- **network_analyzer** — VPC Flow Logs analysis, Security Group rule validation
- **reachability_analyzer** — VPC Reachability Analyzer path analysis
- **tgw_analyzer** — Transit Gateway routing analysis
- **vpn_analyzer** — Site-to-Site VPN status and tunnel monitoring
- **firewall_analyzer** — Network Firewall rule analysis
- **eks_analyzer** — EKS cluster status, node groups, add-ons analysis
- **ecs_analyzer** — ECS services, tasks, container insights
- **istio_analyzer** — Istio service mesh configuration analysis
- **dynamodb_analyzer** — DynamoDB table capacity, GSI/LSI analysis
- **rds_analyzer** — RDS instance performance, parameter group analysis
- **elasticache_analyzer** — ElastiCache cluster status analysis
- **msk_analyzer** — MSK broker status, topic analysis
- **iam_analyzer** — IAM policy simulation, permission analysis
- **cloudwatch_analyzer** — CloudWatch metrics, alarm analysis
- **cloudtrail_analyzer** — CloudTrail event search
- **cost_analyzer** — Cost Explorer cost analysis, forecasting
- **budget_analyzer** — AWS Budgets monitoring
- **cfn_analyzer** — CloudFormation stack analysis, drift detection
- **steampipe_query** — Steampipe SQL query execution

### Dynamic Routing

Automatically routes user questions to the appropriate Gateway.

- `agent.py`: Gateway selection via `payload.gateway` parameter
- `route.ts`: 9-level priority keyword-based routing
- Sonnet intent classifier introduced (`c56f557`) — Analyzes natural language questions to select the optimal Gateway

### AI Testing

Tests conducted to verify the effectiveness of Gateway separation.

- Routing accuracy verified with 104-question test script
- **54% speed improvement** for container-related responses after Gateway separation
- Enhanced response quality through domain-specific tool utilization
