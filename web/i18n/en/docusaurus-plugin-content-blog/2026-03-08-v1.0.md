The first release of AWSops Dashboard. An integrated AWS/Kubernetes operations dashboard built with Next.js 14 + Steampipe + Amazon Bedrock AgentCore.

<!--truncate-->

### CDK Infrastructure

Transitioned from CloudFormation to AWS CDK for infrastructure as code management.

- `infra-cdk/lib/awsops-stack.ts` — VPC, EC2 (t4g.2xlarge ARM64), ALB, CloudFront configuration
- `00-deploy-infra.sh` CDK deployment script (migrated from CloudFormation)
- CDK bootstrap: ap-northeast-2 + us-east-1 regions

### Cognito Authentication

Built a user authentication system using AWS Cognito.

- User Pool + OAuth2 Authorization Code flow
- Lambda@Edge (Python 3.12, us-east-1) — JWT verification at CloudFront viewer-request
- `07-setup-cloudfront-auth.sh` setup script

### AgentCore Runtime + Gateway

AI agent system leveraging Amazon Bedrock AgentCore.

- Strands Agent Framework (arm64 Docker, ECR)
- MCP protocol-based Gateway
- Code Interpreter (`awsops_code_interpreter`)
- 4 sub-step scripts: `06a-setup-agentcore-runtime.sh`, `06b-setup-agentcore-gateway.sh`, `06c-setup-agentcore-tools.sh`, `06d-setup-agentcore-interpreter.sh`

### Initial Dashboard

Core features for AWS resource monitoring.

- 21 pages + 5 API routes
- Steampipe embedded PostgreSQL (380+ AWS tables, 60+ K8s tables)
- Recharts metrics visualization, React Flow network topology
- CIS v1.5~v4.0 benchmark compliance checks
- AI routing: Code Interpreter → AgentCore → Steampipe+Bedrock → Bedrock Direct

### Claude Code Scaffolding

Claude Code integration environment for improved development productivity.

- Module CLAUDE.md — Context documents for each directory
- Auto-Sync Rules — Automatic documentation sync on code changes
- `.claude/hooks/check-doc-sync.sh` — Documentation sync verification hook
