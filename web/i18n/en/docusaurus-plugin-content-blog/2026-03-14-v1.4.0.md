New AgentCore dashboard page, CloudWatch metrics tables for 4 services, Resource Inventory (PR #1).

<!--truncate-->

### AgentCore Dashboard (`/agentcore`)

- Runtime status, 8 Gateway cards, full list of 125 tools
- AgentCore API (`/api/agentcore`): Query Runtime/Gateway status via AWS CLI
- Architecture diagram

### Sign Out

- Logout button in Header (deletes HttpOnly cookie, triggers Cognito re-authentication)
- Cognito demo account: `demo@awsops.local`

### CloudWatch Metrics Tables (4 Services)

- **MSK**: Per-broker CPU/Memory/BytesIn/BytesOut
- **RDS**: Per-instance CPU/Memory/Connections/IOPS
- **ElastiCache**: Per-node CPU/Memory/Network/Connections + Valkey engine card
- **OpenSearch**: Per-domain CPU/JVM Memory/Cluster Status/Search Rate
- All follow the same pattern: `cloudwatch get-metric-data` with progress bar + numeric values

### Resource Inventory (PR #1)

- `/inventory` page: Quantity trends for 18 resource types
- Multi-line chart, cost impact estimation
- Cost Explorer availability management: Automatic MSP Payer detection

### S3/RDS/ElastiCache Enhancements

- **S3**: Bucket Map (TreeMap), search/filter, IAM Roles
- **RDS**: Security Groups, CloudWatch mini charts, Features
- **ElastiCache**: Security Groups, CloudWatch metrics

### Monitoring Enhancements

- Instance detailed metrics view (Date Range 1h~30d)
- Resource Topology: Graph/Map View toggle
