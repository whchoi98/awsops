85 commits, complete UX/feature upgrade. Complex queries now trigger **1-3 Gateway parallel calls** + Bedrock response synthesis.

<!--truncate-->

### Multi-Route AI

- Complex queries → 1-3 Gateway parallel calls + Bedrock response synthesis
- Example: "Analyze VPC security groups and costs" → network + cost parallel calls
- SSE: Multi-route → N Gateways in parallel → response synthesis

### Dashboard Redesign

- 18 cards (6x3 layout), 1:1 sidebar mapping
- Card sub-metrics (e.g., EC2 `18 running · 4 stopped`)
- CIS Compliance Pass Rate, Monthly Cost metrics
- Resource Distribution + K8s Pod Status charts

### AI Assistant Improvements

- SSE streaming: Analyzing → Gateway connection → Tool calls → Complete
- Response time display, clipboard copy, related question suggestions

### New Pages

- CloudFront (`/cloudfront-cdn`), WAF (`/waf`), ECR (`/ecr`)

### VPC & Network Enhancements

- VPC Resource Map: AWS Console-style 4-column layout (VPC → Subnets → RT → Connections)
- Route Table tab, TGW Route Tables, TGW Attachment details
- Resource Topology: Graph View / Map View toggle

### EKS Enhancements

- K8s Overview node cards: CPU/Memory progress bars
- Node details: ENI card + per-ENI traffic + Pods table
- EKS Explorer: Status/Node filters, pagination

### Cost Explorer Enhancements

- Period/Service filters, Projected end-of-month cost, MoM Change
- Monthly Trend line, Cost by Service pie chart
