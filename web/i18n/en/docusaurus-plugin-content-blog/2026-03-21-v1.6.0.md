This release adds **multilingual (Korean/English) support** and a **Bedrock monitoring dashboard**, along with a Docusaurus-based user guide site.

<!--truncate-->

### Internationalization (i18n)

- React Context + localStorage based (no URL routing to avoid basePath conflicts)
- `LanguageContext.tsx`: Provider + useLanguage hook implementation
- `translations/en.json`, `ko.json`: 500+ keys translated each
- Sidebar EN/KO toggle button, default language: Korean
- All 35 pages + Header/Sidebar + AI status messages translated
- AI responses output in English/Korean based on language setting

### Bedrock Monitoring (`/bedrock`)

- Model-specific usage dashboard (CloudWatch + AWSops token tracking)
- Account Total vs AWSops usage comparison chart
- Token cost display in AI chat (input/output tokens, USD cost)
- Prompt caching hit rate display, default range 7 days

### Docusaurus User Guide

- `web/` directory: Korean (default) + English i18n support
- Automated screenshot generation (Playwright + multi-DPR)
- GitHub Pages deployment (`gh-pages` branch)

### Changes

- Dashboard 35 pages, 50 routes structure
- PR #4: Multi-DPR screenshots
- PR #5: Docusaurus guide site
