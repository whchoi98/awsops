# AI 어시스턴트 빠른 시작

AWSops AI 어시스턴트는 Amazon Bedrock AgentCore를 기반으로 자연어로 AWS 인프라에 대해 질문하고 분석을 요청할 수 있는 기능입니다.

## 시작하기

### 1. AI Assistant 페이지 접속

사이드바에서 **AI Assistant**를 클릭합니다.

### 2. 질문 입력

화면 하단의 입력창에 질문을 입력하고 **Enter** 또는 전송 버튼을 클릭합니다.

### 3. 응답 확인

AI가 질문을 분석하여 적절한 Gateway로 라우팅하고, 도구를 활용하여 답변을 생성합니다.

## 예시 질문

### 리소스 현황 조회

```
EC2 인스턴스 현황을 알려줘
```

```
S3 버킷이 몇 개 있는지 확인해줘
```

```
Lambda 함수 목록을 보여줘
```

### 네트워크 분석

```

```

```

```

### 보안 점검

```

```

```
IAM 사용자 중 MFA가 설정되지 않은 계정이 있는지 확인해줘
```

### 비용 분석

```
이번 달 비용 현황을 보여줘
```

```

```

### 컨테이너 현황

```

```

```

```

## 10단계 라우팅

AI 어시스턴트는 질문을 분석하여 10개의 전문 라우트 중 가장 적합한 곳으로 자동 분류합니다.

| 우선순위 | 라우트 | 용도 |
|---------|--------|------|
| 1 | code | Python 코드 실행, 계산, 시각화 |
| 2 | network | VPC, TGW, VPN, Flow Logs 분석 |
| 3 | container | EKS, ECS, Istio 트러블슈팅 |
| 4 | iac | CDK, CloudFormation, Terraform |
| 5 | data | DynamoDB, RDS, ElastiCache, MSK |
| 6 | security | IAM, 정책 시뮬레이션, 보안 요약 |
| 7 | monitoring | CloudWatch, CloudTrail |
| 8 | cost | 비용 분석, 예측, 예산 |
| 9 | aws-data | 리소스 목록/현황 (Steampipe SQL) |
| 10 | general | 일반 AWS 질문, 문서 검색 |

:::tip 라우트 확인
응답 하단에 표시되는 라우트 정보로 어떤 Gateway가 사용되었는지 확인할 수 있습니다.
예: `Network Gateway (17 tools)`, `Bedrock + Steampipe SQL`
:::

## 응답 이해하기

### 응답 구성

```
┌────────────────────────────────────────────────────────┐
│  [AI 아이콘]                                           │
│                                                        │
│  응답 내용 (마크다운 형식)                             │
│  - 테이블, 리스트, 코드 블록 지원                      │
│                                                        │
├────────────────────────────────────────────────────────┤
│  Network Gateway (17 tools)  │  Claude sonnet-4.6  │ 3.2s │
├────────────────────────────────────────────────────────┤
│  Tools: list_vpcs, get_vpc_network_details, ...        │
│  Queried: aws_vpc, aws_vpc_subnet                      │
└────────────────────────────────────────────────────────┘
```

### 표시 정보

- **라우트 경로**: 어떤 Gateway가 처리했는지
- **모델**: 사용된 Claude 모델 (Sonnet/Opus)
- **응답 시간**: 처리에 걸린 시간
- **사용된 도구**: 호출된 MCP 도구 목록
- **조회된 리소스**: Steampipe에서 쿼리한 테이블

## 모델 선택

화면 우측 상단의 드롭다운에서 모델을 선택할 수 있습니다:

- **Claude Sonnet 4.6**: 빠른 응답, 일반적인 질문에 적합 (기본값)
- **Claude Opus 4.6**: 복잡한 분석, 심층적인 추론에 적합

## 연관 질문

응답 후 관련된 후속 질문이 버튼 형태로 표시됩니다. 클릭하면 해당 질문이 자동으로 입력됩니다.

예시:
```
[IAM 사용자 목록과 Access Key 상태를 보여줘]
[MFA가 설정되지 않은 사용자가 있는지 확인해줘]
[보안그룹 중 0.0.0.0/0 인바운드가 있는지 확인해줘]
```

## 대화 이력

### 세션 내 이력
현재 세션의 대화 내용은 화면에 유지됩니다. 이전 대화를 참조하여 후속 질문이 가능합니다.

### 저장된 이력
화면 하단의 **대화 이력** 패널을 펼치면 이전 세션의 대화 기록을 확인하고 다시 질문할 수 있습니다.
