## 다국어 지원 (i18n)

React Context + localStorage 기반으로 URL 라우팅 없이 언어를 전환합니다. Next.js의 `basePath: '/awsops'`와 충돌하지 않는 설계입니다.

<I18nArchitecture />

| 항목 | Before (v1.5) | After (v1.6.0) |
|------|---------------|-----------------|
| 언어 | 한국어만 | 한국어 + 영어 |
| 번역 키 | 0 | **500+** (각 언어) |
| 언어 전환 | 없음 | Sidebar EN/한 토글 버튼 |
| AI 응답 언어 | 한국어만 | 설정에 따라 한/영 자동 전환 |
| URL 변경 | - | 없음 (basePath 충돌 방지) |

:::info 왜 URL 기반 i18n을 사용하지 않는가?
Next.js의 `basePath: '/awsops'`가 설정되어 있어, Next.js 내장 i18n의 locale prefix(`/ko/awsops`, `/en/awsops`)와 충돌합니다. React Context + localStorage 방식은 URL을 변경하지 않아 이 문제를 우회합니다.
:::

**번역 범위**:
- 모든 35개 페이지 텍스트
- Header, Sidebar 메뉴 레이블
- AI 상태 메시지 (분석 중, Gateway 연결 중, 완료 등)
- DataTable 컬럼 헤더
- StatsCard 레이블

## Bedrock 모니터링 (`/bedrock`)

Bedrock 모델별 사용량을 CloudWatch 메트릭과 AWSops 앱 자체 토큰 추적으로 모니터링합니다.

<BedrockMetricsFlow />

| 기능 | 설명 |
|------|------|
| **모델별 사용량** | Claude Sonnet, Opus 등 모델별 호출 수/토큰 |
| **Account vs AWSops** | AWS 계정 전체 Bedrock 사용량 대비 AWSops 앱 비율 |
| **AI 토큰 비용** | 채팅 응답에 입력/출력 토큰 수 + USD 비용 표시 |
| **프롬프트 캐싱** | 캐시 히트율 표시, 비용 절감 효과 확인 |
| **기본 범위** | 7일 (변경 가능) |
| **API 라우트** | `/api/bedrock-metrics` (`src/app/api/bedrock-metrics/route.ts`) |

:::tip AI 토큰 비용 표시
채팅 응답 하단에 `입력: 2,450 토큰 · 출력: 1,200 토큰 · $0.0234` 형태로 비용을 표시합니다.
사용자가 각 질문의 비용을 실시간으로 확인할 수 있어 비용 인식을 높입니다.
:::
