**다국어(한/영) 지원**과 **Bedrock 모니터링 대시보드**를 추가하고, Docusaurus 기반 사용자 가이드 사이트를 구축한 릴리스입니다.

<!--truncate-->

### 다국어 지원 (i18n)

- React Context + localStorage 기반 (URL 라우팅 없음, basePath 충돌 방지)
- `LanguageContext.tsx`: Provider + useLanguage 훅 구현
- `translations/en.json`, `ko.json`: 각 500+ 키 번역
- Sidebar EN/한 토글 버튼, 기본 언어: 한국어
- 모든 35개 페이지 + Header/Sidebar + AI 상태 메시지 번역 완료
- AI 응답이 언어 설정에 따라 영어/한국어로 출력

### Bedrock 모니터링 (`/bedrock`)

- 모델별 사용량 대시보드 (CloudWatch + AWSops 토큰 추적)
- Account Total vs AWSops 사용량 비교 차트
- AI 채팅에 토큰 비용 표시 (입력/출력 토큰, USD 비용)
- 프롬프트 캐싱 히트율 표시, 기본 범위 7일

### Docusaurus 사용자 가이드

- `web/` 디렉토리: 한국어(기본) + 영문 i18n 지원
- 스크린샷 자동 생성 (Playwright + multi-DPR)
- GitHub Pages 배포 (`gh-pages` 브랜치)

### 변경 사항

- 대시보드 35 페이지, 50 라우트 체계
- PR #4: Multi-DPR 스크린샷
- PR #5: Docusaurus 가이드 사이트
