AWSops 대시보드의 첫 릴리스입니다. Next.js 14 + Steampipe + Amazon Bedrock AgentCore로 구축된 **AWS/Kubernetes 통합 운영 대시보드**가 탄생했습니다.

<!--truncate-->

:::tip 핵심 수치
**21개 페이지** · **5개 API 라우트** · **380+ AWS 테이블** · **60+ K8s 테이블** · **125 MCP 도구**
:::

## 아키텍처 개요

CloudFront에서 사용자 요청을 받아 ALB를 거쳐 EC2에서 Next.js 앱이 실행되고, Steampipe를 통해 실시간으로 AWS API를 SQL로 조회합니다.

## CDK 인프라 전환

CloudFormation YAML 템플릿에서 AWS CDK TypeScript로 전환하여 인프라를 코드로 관리합니다.

| 항목 | Before (CloudFormation) | After (CDK) |
|------|------------------------|-------------|
| 언어 | YAML 2000+ 줄 | TypeScript 타입 안전 |
| 배포 | `aws cloudformation deploy` | `cdk deploy` |
| VPC | 수동 서브넷 CIDR 계산 | `ec2.Vpc()` 자동 |
| ALB | 리스너/타겟 수동 | `elbv2.ApplicationLoadBalancer()` |
| 재사용 | 복사-붙여넣기 | Stack 상속 + Construct |

:::info 핵심 파일
- `infra-cdk/lib/awsops-stack.ts` — VPC, EC2(t4g.2xlarge ARM64), ALB, CloudFront 구성
- `00-deploy-infra.sh` — CDK bootstrap (`ap-northeast-2` + `us-east-1`) 및 배포 스크립트
:::

## Cognito 인증

AWS Cognito를 활용한 사용자 인증 시스템을 구축했습니다.

- **Lambda@Edge** (Python 3.12, us-east-1) — CloudFront viewer-request에서 JWT 검증
- **OAuth2 Authorization Code** flow로 안전한 인증
- `07-setup-cloudfront-auth.sh` 스크립트로 자동 설정

## AgentCore Runtime + Gateway

Amazon Bedrock AgentCore를 활용한 AI 에이전트 시스템입니다.

<AgentCoreFlow />

| 구성 요소 | 설명 |
|-----------|------|
| Strands Agent | arm64 Docker 이미지, ECR에서 관리 |
| MCP 프로토콜 | Gateway → Lambda 간 표준 통신 |
| Code Interpreter | `awsops_code_interpreter` (Python sandbox) |
| 배포 스크립트 | `06a` Runtime, `06b` Gateway, `06c` Tools, `06d` Interpreter |

:::warning Docker 빌드 주의사항
EC2에서는 Docker 이미지 **빌드만** 수행합니다. 실행은 AgentCore 관리형 서비스에서 컨테이너로 실행됩니다.
arm64 필수: `docker buildx --platform linux/arm64 --load`
:::
