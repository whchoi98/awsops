AWSops 대시보드의 첫 릴리스. Next.js 14 + Steampipe + Amazon Bedrock AgentCore로 구축된 AWS/Kubernetes 통합 운영 대시보드.

<!--truncate-->

### CDK 인프라

CloudFormation에서 AWS CDK로 전환하여 인프라를 코드로 관리합니다.

- `infra-cdk/lib/awsops-stack.ts` — VPC, EC2(t4g.2xlarge ARM64), ALB, CloudFront 구성
- `00-deploy-infra.sh` CDK 배포 스크립트 (CloudFormation에서 전환)
- CDK bootstrap: ap-northeast-2 + us-east-1 리전

### Cognito 인증

AWS Cognito를 활용한 사용자 인증 시스템을 구축했습니다.

- User Pool + OAuth2 Authorization Code flow
- Lambda@Edge (Python 3.12, us-east-1) — CloudFront viewer-request에서 JWT 검증
- `07-setup-cloudfront-auth.sh` 설정 스크립트

### AgentCore Runtime + Gateway

Amazon Bedrock AgentCore를 활용한 AI 에이전트 시스템입니다.

- Strands Agent Framework (arm64 Docker, ECR)
- MCP 프로토콜 기반 Gateway
- Code Interpreter (`awsops_code_interpreter`)
- 4개 하위 단계 스크립트: `06a-setup-agentcore-runtime.sh`, `06b-setup-agentcore-gateway.sh`, `06c-setup-agentcore-tools.sh`, `06d-setup-agentcore-interpreter.sh`

### 초기 대시보드

AWS 리소스 모니터링을 위한 핵심 기능을 제공합니다.

- 21개 페이지 + 5개 API 라우트
- Steampipe 내장 PostgreSQL (380+ AWS 테이블, 60+ K8s 테이블)
- Recharts 메트릭 시각화, React Flow 네트워크 토폴로지
- CIS v1.5~v4.0 벤치마크 컴플라이언스 체크
- AI 라우팅: Code Interpreter → AgentCore → Steampipe+Bedrock → Bedrock Direct

### Claude Code 스캐폴딩

개발 생산성 향상을 위한 Claude Code 통합 환경을 구성했습니다.

- Module CLAUDE.md — 각 디렉토리별 컨텍스트 문서
- Auto-Sync Rules — 코드 변경 시 문서 자동 동기화
- `.claude/hooks/check-doc-sync.sh` — 문서 동기화 검증 훅
