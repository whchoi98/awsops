AgentCore 대시보드 페이지 신규 추가, 4개 서비스 CloudWatch 메트릭 테이블, Resource Inventory(PR #1).

<!--truncate-->

### AgentCore 대시보드 (`/agentcore`)

- Runtime 상태, 8 Gateway 카드, 125 도구 전체 목록
- AgentCore API (`/api/agentcore`): AWS CLI로 Runtime/Gateway 상태 조회
- 아키텍처 다이어그램

### Sign Out

- Header에 로그아웃 버튼 (HttpOnly 쿠키 삭제 → Cognito 재인증)
- Cognito demo 계정: `demo@awsops.local`

### CloudWatch 메트릭 테이블 (4개 서비스)

- **MSK**: 브로커별 CPU/Memory/BytesIn/BytesOut
- **RDS**: 인스턴스별 CPU/Memory/Connections/IOPS
- **ElastiCache**: 노드별 CPU/Memory/Network/Connections + Valkey 엔진 카드
- **OpenSearch**: 도메인별 CPU/JVM Memory/Cluster Status/Search Rate
- 모두 동일 패턴: `cloudwatch get-metric-data` → 프로그레스 바 + 수치

### Resource Inventory (PR #1)

- `/inventory` 페이지: 18종 리소스 수량 추이
- 멀티라인 차트, 비용 영향 추정
- Cost Explorer 가용성 관리: MSP Payer 자동 판별

### S3/RDS/ElastiCache 강화

- **S3**: Bucket Map (TreeMap), 검색/필터, IAM Roles
- **RDS**: Security Groups, CloudWatch 미니 차트, Features
- **ElastiCache**: Security Groups, CloudWatch 메트릭

### Monitoring 강화

- 인스턴스 상세 메트릭 뷰 (Date Range 1h~30d)
- Resource Topology: Graph/Map View 전환
