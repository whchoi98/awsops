85개 커밋, 전면 UX/기능 업그레이드. 복합 질문 시 **1~3개 Gateway 병렬 호출** + Bedrock 응답 합성.

<!--truncate-->

### 멀티 라우트 AI

- 복합 질문 → 1~3개 Gateway 병렬 호출 + Bedrock 응답 합성
- 예: "VPC 보안그룹과 비용을 분석해줘" → network + cost 병렬 호출
- SSE: 멀티 라우트 → N개 Gateway 병렬 → 응답 합성

### 대시보드 전면 재설계

- 18개 카드 (6x3 레이아웃), 사이드바 1:1 매핑
- 카드 하단 sub-metrics (예: EC2 `18 running · 4 stopped`)
- CIS Compliance Pass Rate, Monthly Cost 지표
- Resource Distribution + K8s Pod Status 차트

### AI Assistant 개선

- SSE 스트리밍: 분석 중 → Gateway 연결 → 도구 호출 → 완료
- 응답 시간 표시, 클립보드 복사, 연관 추천 질문

### 새 페이지

- CloudFront (`/cloudfront-cdn`), WAF (`/waf`), ECR (`/ecr`)

### VPC & Network 강화

- VPC Resource Map: AWS 콘솔 스타일 4컬럼 (VPC → Subnets → RT → Connections)
- Route Table 탭, TGW Route Tables, TGW Attachment 상세
- Resource Topology: Graph View / Map View 전환

### EKS 강화

- K8s Overview 노드 카드: CPU/Memory 프로그레스 바
- 노드 상세: ENI 카드 + per-ENI 트래픽 + Pods 테이블
- EKS Explorer: Status/Node 필터, 페이지네이션

### Cost Explorer 강화

- 기간/서비스 필터, Projected 월말 비용, MoM Change
- Monthly Trend 라인, Cost by Service 파이 차트
