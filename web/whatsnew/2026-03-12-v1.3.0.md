85개 커밋에 걸친 전면 UX/기능 업그레이드입니다. 복합 질문 시 **1~3개 Gateway 병렬 호출** + Bedrock 응답 합성이 핵심입니다.

<!--truncate-->

:::tip 핵심 수치
**85 커밋** · **18개 대시보드 카드** · **멀티 라우트 (1~3 Gateway 병렬)** · **SSE 실시간 스트리밍** · **3개 신규 페이지**
:::

## 멀티 라우트 AI 아키텍처

복합 질문을 분석하여 최대 3개 Gateway를 **동시에** 호출하고, Bedrock이 응답을 합성합니다.

<MultiRouteSequence />

| 항목 | Before (v1.2.0) | After (v1.3.0) |
|------|-----------------|-----------------|
| 질문 처리 | 단일 Gateway만 호출 | 1~3개 Gateway 병렬 호출 |
| 복합 질문 | 첫 번째 매칭 라우트만 사용 | 모든 관련 라우트 병렬 처리 |
| 응답 합성 | 없음 | Bedrock `synthesizeResponses()` |
| 스트리밍 | 없음 | SSE `chunk` 이벤트 실시간 전송 |

:::info 멀티 라우트 동작 원리
`src/app/api/ai/route.ts`의 인텐트 분류기가 질문을 분석하여 `routes: string[]` 배열을 반환합니다.
배열 길이가 2 이상이면 `Promise.allSettled()`로 병렬 호출하고, 성공한 응답들을 `synthesizeResponses()`로 합성합니다.
:::

## 대시보드 전면 재설계

18개 카드를 6x3 레이아웃으로 배치하고, 사이드바 메뉴와 1:1 매핑했습니다.

<DashboardCardsLayout />

- **카드 하단 sub-metrics**: `18 running · 4 stopped` 형태로 핵심 수치 표시
- **CIS Compliance Pass Rate**: 전체 벤치마크 통과율 게이지
- **Monthly Cost**: 이번 달 비용 + MoM 변화율
- **Resource Distribution + K8s Pod Status**: Recharts 차트로 시각화

## SSE 스트리밍 AI 응답

AI 응답을 실시간으로 스트리밍하여 UX를 개선했습니다.

<SSEEventsFlow />

- **`status` 이벤트**: 분석 중 → Gateway 연결 → 도구 호출 단계 표시
- **`chunk` 이벤트**: 응답 텍스트 실시간 전송
- **`done` 이벤트**: 최종 결과 + 사용된 도구 + 응답 시간 + 토큰 수
- 응답 시간 표시, 클립보드 복사, 연관 추천 질문 기능 추가

## 신규 페이지 (3개)

| 페이지 | 경로 | 주요 기능 |
|--------|------|-----------|
| CloudFront | `/cloudfront-cdn` | Distribution 목록, Origin, Behavior, 캐시 정책 |
| WAF | `/waf` | WebACL 규칙, IP Set, 차단 로그 |
| ECR | `/ecr` | 리포지토리, 이미지 태그, 취약점 스캔 결과 |

## VPC & Network 강화

- **VPC Resource Map**: AWS 콘솔 스타일 4컬럼 레이아웃
  ```
  VPC → Subnets → Route Tables → Connections (IGW/NAT/TGW/VPN)
  ```
- **Route Table 탭**: 서브넷 연결, 라우팅 규칙 테이블
- **TGW Route Tables**: Transit Gateway 라우트 테이블 상세
- **Resource Topology**: React Flow 기반 Graph View / Map View 전환

## EKS & Cost 강화

**EKS 개선**:
- K8s Overview 노드 카드: CPU/Memory 프로그레스 바
- 노드 상세: ENI 카드 + per-ENI 트래픽 + Pods 테이블
- EKS Explorer: Status/Node 필터, 페이지네이션

**Cost Explorer 개선**:
- 기간/서비스 필터, Projected 월말 비용, MoM Change
- Monthly Trend 라인, Cost by Service 파이 차트

:::warning Steampipe Cost 쿼리 주의
Cost Explorer 데이터는 **MSP(Managed Service Provider) Payer 계정**에서만 사용 가능합니다.
`checkCostAvailability()`로 사전 확인하고, 불가능하면 `cost-snapshot.ts` 폴백 데이터를 사용합니다.
:::
