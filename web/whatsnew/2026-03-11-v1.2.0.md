7 → **8 Gateway** 분리와 배포 안정성 대폭 개선. Container 응답 54% 속도 개선(50s → 23s), 전체 테스트 104/104 통과.

### Gateway 분리: Infra → Network + Container

- Infra Gateway (41 tools) → Network (17) + Container (24)
- Container 질문 응답 54% 개선 (50s → 23s)
- 8 Gateways, 10 Routes 체계 확립

### 배포 안정성 개선

- `06a`: `run_or_fail` 헬퍼 + AWS 자격 증명 프리플라이트 체크
- `06b`: 8 Gateways 생성
- `07`: CloudFront 자동 감지 — CDK 스택 우선, ALB origin 폴백
- Steampipe 비밀번호 하드코딩 → 동적 조회

### 테스트 결과

| 항목 | 결과 |
|------|------|
| 질문 통과 | 104/104 |
| 라우트 분류 정확도 | 93% (97/104) |

### 수정사항

- EKS access entry: `arn:aws:sts::` → `arn:aws:iam::` 변환
- Cognito custom domain: `SupportedIdentityProviders` 누락
- AgentCore `bedrock-agentcore:*` 권한 추가
- k8s.ts: PVC JSONB 직렬화 에러 → `::text` 캐스팅
