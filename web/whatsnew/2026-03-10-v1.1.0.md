단일 Gateway(29개 도구)에서 **7개 전문 Gateway × 125 MCP 도구**로 전면 재설계한 마일스톤 릴리스입니다.

<!--truncate-->

### 7개 역할 기반 Gateway

각 Gateway는 특정 도메인에 최적화된 도구 세트를 제공합니다.

| Gateway | 도구 수 | 주요 기능 |
|---------|---------|-----------|
| Infra | 41 | Network, EKS, ECS, Istio |
| IaC | 12 | CloudFormation, CDK, Terraform |
| Data | 24 | DynamoDB, RDS, ElastiCache, MSK |
| Security | 14 | IAM 사용자/역할/정책/시뮬레이션 |
| Monitoring | 16 | CloudWatch, CloudTrail |
| Cost | 9 | Cost Explorer, Budgets |
| Ops | 9 | Steampipe SQL, AWS 문서 |

### 19개 Lambda 함수

MCP 도구를 제공하는 Lambda 함수 목록입니다.

- **network_analyzer** — VPC Flow Logs 분석, Security Group 규칙 검증
- **reachability_analyzer** — VPC Reachability Analyzer 경로 분석
- **tgw_analyzer** — Transit Gateway 라우팅 분석
- **vpn_analyzer** — Site-to-Site VPN 상태 및 터널 모니터링
- **firewall_analyzer** — Network Firewall 규칙 분석
- **eks_analyzer** — EKS 클러스터 상태, 노드 그룹, 애드온 분석
- **ecs_analyzer** — ECS 서비스, 태스크, 컨테이너 인사이트
- **istio_analyzer** — Istio 서비스 메시 구성 분석
- **dynamodb_analyzer** — DynamoDB 테이블 용량, GSI/LSI 분석
- **rds_analyzer** — RDS 인스턴스 성능, 파라미터 그룹 분석
- **elasticache_analyzer** — ElastiCache 클러스터 상태 분석
- **msk_analyzer** — MSK 브로커 상태, 토픽 분석
- **iam_analyzer** — IAM 정책 시뮬레이션, 권한 분석
- **cloudwatch_analyzer** — CloudWatch 메트릭, 알람 분석
- **cloudtrail_analyzer** — CloudTrail 이벤트 검색
- **cost_analyzer** — Cost Explorer 비용 분석, 예측
- **budget_analyzer** — AWS Budgets 예산 모니터링
- **cfn_analyzer** — CloudFormation 스택 분석, 드리프트 감지
- **steampipe_query** — Steampipe SQL 쿼리 실행

### 동적 라우팅

사용자 질문을 적절한 Gateway로 자동 라우팅합니다.

- `agent.py`: `payload.gateway` 파라미터로 게이트웨이 선택
- `route.ts`: 9단계 우선순위 키워드 기반 라우팅
- Sonnet 인텐트 분류기 도입 (`c56f557`) — 자연어 질문을 분석하여 최적의 Gateway 선택

### AI 테스트

Gateway 분리 효과를 검증하기 위한 테스트를 수행했습니다.

- 104개 질문 테스트 스크립트로 라우팅 정확도 검증
