**AgentCore Memory Store** 도입으로 사용자별 대화 이력 365일 보관. EBS, MSK, OpenSearch 신규 페이지와 ECS/EKS 컨테이너 비용 분석 추가.

<!--truncate-->

### AgentCore Memory Store

- `06e-setup-agentcore-memory.sh` — Memory 생성 (365일 보관)
- 대화 자동 저장: 질문/요약/라우트/도구/응답시간
- 사용자별 분리: Cognito JWT에서 email 추출
- AgentCore 페이지: 호출 통계 (6 StatsCards) + 대화 이력 검색
- AI 페이지 하단: 대화 이력 토글 패널

### 신규 페이지 (4개)

- **EBS** (`/ebs`): 볼륨/스냅샷, 암호화, EC2 어태치먼트, Idle 볼륨 감지
- **MSK** (`/msk`): Kafka 클러스터, 브로커 노드 (CPU/Memory/Network), KRaft
- **OpenSearch** (`/opensearch`): 도메인, 암호화(N2N/At-Rest), VPC, 클러스터 구성
- **Resource Inventory** (`/inventory`): 18종 리소스 수량 추이

### 컨테이너 비용 분석

- `/container-cost`: ECS Fargate 비용 (Container Insights + 가격 계산)
- `/eks-container-cost`: EKS Pod 비용 (OpenCost API + Request 기반 폴백)
- `06f-setup-opencost.sh`: Prometheus + OpenCost 설치

### AI 라우팅 개선

- AgentCore 설정 외부화: `data/config.json`에서 ARN 읽기
- 도구 사용 표시: 키워드 매칭으로 MCP 도구 추론 → UI 배지
- 멀티 라우트 실패 시 Bedrock Direct 폴백, 타임아웃 60s → 90s

### 보안 강화

- Sign Out: HttpOnly 쿠키 서버 사이드 API 삭제
- 하드코딩 값 3개 제거 (password, interpreter ID, gateway URLs)
